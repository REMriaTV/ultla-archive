# 決済ツール選定のセカンドオピニオンを求めるためのプロンプト

以下のテキストを、決済に詳しい専門家や別の相談先にそのまま渡して、フラットな意見を求めるときに使ってください。必要に応じて先頭の【依頼】だけ送ることも、全文を送ることもできます。

---

## 【依頼】

私たちは教育・研修向けコンテンツのサブスクリプションサイトを運営しており、決済手段として **Stripe** の導入を検討しています。すでに Stripe を用いた月額サブスク（Checkout / Customer Portal / Webhook）の実装を一部進めていますが、「この規模・仕様で Stripe でよいか」について、先入観なしのセカンドオピニオンが欲しいです。

**お伺いしたいこと**

- 下記のサービス概要・決済仕様を踏まえ、**月額サブスク課金の決済基盤として Stripe を採用すること**について、メリット・デメリットや注意点を率直に教えてください。
- 同様の要件であれば、**Stripe 以外に検討すべきサービス**（Lemon Squeezy、Paddle、その他）はありますか。ある場合、どのような観点で比較・選定するとよいかも知りたいです。
- 日本国内のユーザーが主で、請求は日本円（JPY）を想定しています。この前提で、選定や契約・運用で気をつけるべき点があれば教えてください。

ご意見は「Stripe を推す／推さない」の結論ではなく、**選択肢と判断材料をフラットに整理していただけると助かります**。

## 【サービス・決済の前提（相談用メモ）】

### サービス概要

- **名称**: SPACE ARCHIVE（学び・研修向けコンテンツのアーカイブ）
- **主なコンテンツ**: プログラムスライド（PDF）、将来的に動画・ワークシート等を想定
- **主な想定ユーザー**: 国内の教育・研修関係者（自治体の食育研修参加者などが想定の一例）
- **技術スタック**: Next.js（App Router）、Supabase（認証・DB・Storage）、Vercel 想定でホスティング

### 課金モデル（確定方針）

- **課金形態**: **サブスクリプション（月額）のみ**。単品購入・レンタルは設けない。
- **プラン構成**:
  - **BASIC**: 無料。ログイン済みユーザー。一定範囲のコンテンツを閲覧可能。招待コードで紐づいたコンテンツも閲覧可能。
  - **PRO**: 有料・月額。想定単価は月額 300〜900 円程度。PRO レベルまでのコンテンツをフル閲覧。
  - **ADVANCE**: 有料・月額。想定単価は月額数千円程度。全コンテンツ閲覧＋ダウンロード可。
- **通貨**: 日本円（JPY）
- **決済のトリガー**: ユーザーがマイページの「プラン登録」から PRO または ADVANCE を選択し、決済ページへ遷移して月額サブスクに加入。解約・支払い方法変更は顧客ポータル（Customer Portal）で行う想定。

### 現在の実装状況（Stripe の場合）

- Stripe 用の **Checkout セッション**（サブスク作成）、**Customer Portal**（解約・支払い方法変更）、**Webhook**（`customer.subscription.created/updated/deleted` で自社DBの `subscriptions` および `profiles.plan` を更新）の API を実装済み。
- 料金は Stripe ダッシュボードで「PRO 用」「ADVANCE 用」の月額料金をそれぞれ作成し、Price ID を環境変数で渡す想定。
- ドキュメント（STRIPE_SETUP.md）で、API キー・Price ID・Webhook 署名シークレットの取得手順と `.env` の設定を整理済み。

### 運営・規模のイメージ

- 運営主体: 株式会社（日本）。小規模〜中規模の利用を想定。
- 現時点では、まずは「PRO / ADVANCE の月額課金が確実に動くこと」を優先。売上規模が大きくなった段階で、手数料や税・請求書まわりを再検討する想定。

### 迷っている点

- **Stripe でよいか**: 実装は進めやすいが、日本法人・日本ユーザー・JPY という前提で、契約・審査・手数料・サポートの面で他ツールと比較した優位性・劣位性を客観的に知りたい。
- **代替案**: Lemon Squeezy や Paddle など、販売者代行型（Merchant of Record）のサービスを選ぶメリット・デメリット（税・請求書・利用規約・ chargeback 対応など）も含めて、比較の観点を教えてもらいたい。
