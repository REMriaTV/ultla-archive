# SPACE ARCHIVE サブスクリプション移行計画

**作成日**: 2026-02-28  
**更新**: 方針確定 — ログイン＝BASIC、ADVANCE に用語統一、招待コード動線を整理

---

## 概要

現在の招待コード制から、**ログイン＝BASIC アカウント**＋**サブスクのみ**の3段階プラン（BASIC / PRO / **ADVANCE**）に移行する。  
個別購入・レンタルは設けず、プランごとに閲覧できるコンテンツレベルを切り替える形でシンプルに運用する。  
**招待コードを持っていれば** BASIC モード（ログインはしてるがサブスク未加入）でもコード紐づきコンテンツを閲覧可能。**招待コードの期限が切れたコンテンツ**や**元からコードを持っていないコンテンツ**を見るには、**サブスク（PRO または ADVANCE）に登録してね**、という動線にする。

---

## 1. プラン構成（確定）

### 3段階: BASIC / PRO / ADVANCE（ADVANCE に用語統一）

| プラン | 条件 | 決済 | 見られる範囲 |
|--------|------|------|--------------|
| **BASIC** | ログイン済み | 不要 | BASIC コンテンツ全ページ ＋ 招待コード紐づきコンテンツ全ページ |
| **PRO** | サブスク加入 | 月額（例: 300〜900円程度） | BASIC ＋ PRO レベルコンテンツ（研修・個人情報含むものは除く） |
| **ADVANCE** | サブスク加入 | 月額（高め） | 全コンテンツ一括 ＋ ダウンロード可 |

- **ログインできたら BASIC アカウント**。決済不要でハードルが低い。
- **招待コードを持っていれば** BASIC モード（サブスクに登録したくない人）でも、**コードに紐づいた特定コンテンツ**が閲覧できる。
- **招待コードの期限が切れた**／**元から招待コードを持っていないコンテンツ**を見たい → **PRO または ADVANCE サブスクに登録**してもらう動線。マイページの招待コードページ（期限切れ時）や、4枚制限オーバーレイなどで「サブスク登録で続きを見る」を案内する。

### ゲスト（未ログイン）

- 一覧: visibility=free のスライドのみ。
- 閲覧: 4 枚まで（現状どおり）。

### 招待コードの扱い

- 招待コードは**現状どおり維持**。ログイン必須。コード紐づきコンテンツは BASIC で閲覧可能（決済不要）。
- **期限切れ時**: マイページの招待コードページにサブスク案内（PRO／ADVANCE）を掲載し、「サブスクに登録してね」へ誘導。サブスク登録画面へリンク。
- サブスク実装を、お試しコードの期限終了日を目安に完了させる。

## 2. コンテンツレベル（スライド側の設定）

スライドは3レベルで登録する（DB: **content_tier** = `basic` / `pro` / `advance`）。**プランごとに閲覧できるレベルを切り替える**形でシンプルに運用。

| レベル | 例 | 閲覧できるプラン |
|--------|-----|------------------|
| **basic** | ログインユーザー向けフリー、昔のスライド・現場でアレンジ推奨 | BASIC 以上 |
| **pro** | 最近のコンテンツ・完成度の高いもの（研修スライド・個人情報含むものは除く） | PRO 以上 |
| **advance** | 研修で使ったスライド、個人情報コンテンツ、ダイジェスト動画など | ADVANCE のみ |

- **PRO** なら一定のスライドがフル閲覧できるサブスク。研修スライドや個人情報コンテンツは見られないが、最近のコンテンツ・完成度の高いコンテンツは個別購入なしで見られる。
- **ADVANCE** なら研修スライドも含め全コンテンツ一括閲覧＋ダウンロード可。

## 3. 新規コンテンツ種別

現在はプログラムスライドのみ。以下を追加予定。

| 種別 | 説明 | セキュリティ |
|------|------|-------------|
| **プログラムダイジェスト動画** | 子供の顔が写る。報告会・信頼できる知人に共有 | 招待コード or ADVANCEのみ |
| **STEAM CHAOS教材動画** | 経産省未来の教室実証事業。全9シリーズ64本。付随ワークシートあり | サブスクで広告なし視聴、ワークシートDL可 |
| **風のカード** | 奈良の保育士向け内省型研修カード。プロトタイプ→有料化 | 購入で利用 |
| **ワークシート** | プログラムスライドの付随資料 | スライドと連動して配布 |

## 4. 価格・運用の考え方

- **個別購入・レンタルは設けない**。値付け・著作権まわり・運用をシンプルにし、**サブスクでプランごとに閲覧レベルを切り替える**形にする。
- PRO: 月額 300〜900 円程度を想定。PRO レベルまでフル閲覧。
- ADVANCE: 月額高め。全コンテンツ一括＋ダウンロード可。
