【1】ユーザー側のプラン 4段階
プラン	状態	決済	ヘッダー表示例
FREE（ゲスト）	未ログイン	なし	FREE（ゲスト）
BASIC	ログイン済み・サブスク未加入	不要	BASIC
PRO	サブスク加入（月額）	月額 300〜900 円程度想定	PRO
ADVANCE	サブスク加入（月額）	月額高め想定	ADVANCE
※ログインした時点で BASIC。有料は PRO と ADVANCE の2段階のみ。個別購入・レンタルは設けない。

【2】コンテンツ側のレベル（content_tier）
content_tier	想定される中身の例	フル閲覧できるプラン
basic	ログインユーザー向けフリー、昔のスライド・現場でアレンジ推奨	BASIC 以上（ログインすれば全ページ可）
pro	最近のコンテンツ・完成度の高いもの（研修・個人情報含むものは除く）	PRO 以上
advance	研修で使ったスライド、個人情報コンテンツ、ダイジェスト動画など	ADVANCE のみ

【3】一覧に「出る／出ない」（visibility）
visibility	誰に一覧表示されるか
free	全員（ゲスト含む）
invite_only	招待コードで紐づいているユーザー、または PRO / ADVANCE ユーザー（有料プランは実質全スライド一覧可）
private	管理者のみ（一般・ゲストには一覧に出さない）

【4】誰が何を全ページ見られるか（対応表）
	ゲスト	BASIC	PRO	ADVANCE
content_tier = basic	4枚まで	全ページ	全ページ	全ページ
content_tier = pro	（一覧に出ない）	4枚まで or 招待紐づきで全ページ	全ページ	全ページ
content_tier = advance	（一覧に出ない）	4枚まで or 招待紐づきで全ページ	4枚まで or 招待紐づきで全ページ	全ページ＋DL可
※招待コードで紐づいたスライドは、content_tier に関係なく該当ユーザーは全ページ閲覧可能。ADVANCE は全コンテンツ全ページ＋DL可。

【5】招待コードの役割
対象	動線・説明
お金を払わず特定コンテンツだけ見たい層	招待コードを発行。コードを入力したユーザーは BASIC のまま、コードに紐づいたスライドだけ全ページ閲覧可能。
招待コード期限切れ／コードがないコンテンツを見たい層	PRO または ADVANCE のサブスクに登録してもらう。マイページ・4枚制限オーバーレイで「サブスク登録で続きを見る」を案内。
ログインの価値	ゲスト＝4枚まで。ログイン＝BASIC コンテンツ全ページ＋招待コード紐づき全ページ（決済なし）。

【6】課金の棲み分け
項目	内容
課金形態	月額サブスクのみ。単品購入・レンタルはなし。
PRO	月額 300〜900 円程度想定。basic ＋ pro コンテンツをフル閲覧。
ADVANCE	月額高め。全コンテンツ一括＋ダウンロード可。
方針	プランごとに閲覧レベルを切り替える形に統一。

【7】用語・DB との対応
項目	内容
表示・案内	BASIC / PRO / ADVANCE に統一（「プレミアム」は使わない）
DB profiles.plan	free | basic | pro | advance（premium は advance と同義）。free はゲストのみ。ログイン時は basic を付与。
DB content_tier	basic / pro / advance の3値。スライド編集・管理画面で設定。
