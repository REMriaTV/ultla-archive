# SPACE ARCHIVE ビジネスシミュレーション＆戦略ガイド

**作成日**: 2026年3月  
**前提**: 月額サブスクリプションモデル（Stripe決済想定）  
**ターゲット**: 教育・研修関係者、自治体、学校法人、保護者など  

> **社外非公開** — 運営チーム・関係者向けの指針です。外部に共有しないでください。いつでもこのドキュメントを参照して方針を揃えてください。

---

## 1. 単価設定の基本方針（仮）

* **PROプラン**: 月額 **500円**
  * *ターゲット*: 個人の教員、ファシリテーター、保護者が自腹で手軽に払いやすい価格帯。
* **ADVANCEプラン**: 月額 **3,000円**
  * *ターゲット*: 熱心な教育実践者、フリースクール運営者。または学校・自治体の「経費・研修費」として落とす価格帯。

---

## 2. 成長フェーズ別 売上シミュレーション

### フェーズ1：スモールスタート（コアファン・過去参加者層）

これまでの研修参加者や、直接つながりのある教育関係者に限定的に案内し、初期の熱量を作っていくフェーズ。

* **リーチ数（見込み客）**: 1,000人
* **PRO登録（想定5%）**: 50人 × 500円 = **25,000円 / 月**
* **ADVANCE登録（想定1%）**: 10人 × 3,000円 = **30,000円 / 月**
* **合計月間売上**: **55,000円** （年間見込：約66万円）
* **マイルストーン**: サーバー代（Vercel, Supabase等）のランニングコストを黒字化し、初期ユーザーからのフィードバックを得てUI/UXを改善する。

### フェーズ2：ミドル展開（口コミ拡大・新規セミナー経由）

定期的なセミナー開催と、「招待コード」を活用したフリーミアム戦略（無料枠からのアップセル）が機能し始めたフェーズ。

* **リーチ数（見込み客）**: 5,000人
* **PRO登録（想定10%）**: 500人 × 500円 = **250,000円 / 月**
* **ADVANCE登録（想定2%）**: 100人 × 3,000円 = **300,000円 / 月**
* **合計月間売上**: **550,000円** （年間見込：約660万円）
* **マイルストーン**: 安定した事業としての柱が確立。ADVANCEプランの契約数が売上の牽引役となる。

### フェーズ3：事業拡大期（BtoB・施設単位の導入）

個人だけでなく、自治体の教育委員会、フリースクール、民間学童などが「施設単位（法人枠）」でADVANCEプランを一括契約し始めたフェーズ。

* **リーチ数（見込み客）**: 20,000人
* **PRO登録（想定10%）**: 2,000人 × 500円 = **1,000,000円 / 月**
* **ADVANCE登録（法人含む3%）**: 600人 × 3,000円 = **1,800,000円 / 月**
* **合計月間売上**: **2,800,000円** （年間見込：約3,360万円）
* **マイルストーン**: 教育系コンテンツ特有の「低い解約率（チャーンレート）」を活かし、強固なストックビジネスとして自走する。

---

## 3. 売上最大化と事業戦略のポイント

### ① 「招待コード」を起点としたフリーミアム戦略

* セミナー参加者限定の「招待コード」は見込み客（リスト）を集める最強のフック。
* コードで一部のコンテンツを無料で体験させ、「他のスライドも見たい」「全ページダウンロードして授業で使いたい」という熱量が高まったタイミングで PRO / ADVANCE への登録動線を設計する。

### ② BtoB（法人・自治体）向け決済の準備

* ADVANCEプラン（月数千円〜）は、個人の財布からは出にくくても「学校の教材費」「自治体の研修費」としては極めて安価。
* 将来的には、クレジットカード（Stripe Checkout）だけでなく、「年間一括の請求書払い（銀行振込）」での契約ルートを整備することで、ADVANCE層の獲得を加速させる。

### ③ 高利益率の再投資

* デジタルアーカイブのサブスクモデルは、利益率が極めて高い（Stripe手数料約3.6%＋サーバー代等を除いた大部分が粗利）。
* 得られた利益を、UI改修、新規プログラムの動画撮影、ワークシートの開発などに再投資し、コンテンツの価値（LTV）を高め続けるサイクルを回す。

## 4. プラン開始時の周知方針（葉山との関係）

- 現時点では、SPACE ARCHIVE の有料プラン（PRO / ADVANCE）は **プロトタイピング段階**であり、正式開始の時期を対外的に約束できる状況ではない。  
- そのため、葉山町の参加者に対して、現段階で **「プラン開始通知メールの登録」や「事前の一斉メール案内」などは行わない。**  
- プラン機能と運用方針が固まり、
  1. プロトタイプが完成  
  2. 福本理恵さんによる確認・了承  
  3. 本番サイトでの実装が完了  
  したタイミングで、**まず葉山町の担当者にメールで情報提供する**。  
- その上で、参加者への案内方法（メール配信・学校経由の周知など）は、**葉山町の担当者側の判断・運用に委ねる。**  
- サイト側では、プラン開始時に「お知らせ（announcements）」を出し、トップページ・マイページからの導線で周知する。
