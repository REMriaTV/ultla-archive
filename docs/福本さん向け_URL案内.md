# 福本さんへ：スライドアーカイブのURL案内（テンプレート）

DNS 反映後、この内容をメールやチャットでそのまま（または少し編集して）送れます。

---

## 案内文（コピー用）

```
福本先生

いつもお世話になっております。
スライドアーカイブのサイトが公開できましたので、URLをお知らせします。

【スライドアーカイブ】
https://space-archive.jp

※ https://www.space-archive.jp でも同じサイトにアクセスできます。

■ 使い方
・上記URLをブラウザのアドレス欄に入力するか、ブックマーク（お気に入り）に追加しておくと便利です。
・初回は「Google でログイン」またはメールアドレスで会員登録が必要です。
・ログイン時に「supabase.co」という長いアドレスが一瞬表示されることがありますが、認証のための安全な処理ですのでそのまま進めてください。

■ よくある質問
・招待コードをお持ちの場合は、ログイン後「マイページ」→「招待コード」から入力できます。
・スライドを「マイリスト」に保存すると、あとからすぐ見つけられます。

不明点があればお気軽にご連絡ください。
```

---

## 伝えるときのポイント

- **URLは1つだけ伝える**：「https://space-archive.jp」だけでOK（www ありは言わなくてよい）。
- **ブックマークを勧める**：先生方の端末で「お気に入り」に追加してもらうと、次回から迷いにくい。
- **ログイン画面の説明**：事前に「supabase.co と出ても大丈夫」と一言あると安心してもらいやすい（ログイン画面にも短い説明文を入れ済み）。
- **問い合わせ先**：困ったときの連絡先（メール or 担当者名）を文末に書いておくと親切。

---

## 編集のヒント

- 挨拶や敬称は普段のやり取りに合わせて調整してください。
- 「株式会社SPACE 運営」など、運営主体を入れたい場合は1行足してOKです。
- 招待コードの配布方法が決まっていれば、その案内を追加してもよいです。
